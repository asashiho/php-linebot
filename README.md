LINE BOT×画像認識APIを使ったBOT APIサンプル
======================
LINE BOT APIと画像認識APIを使って
画像認識を行うためのサンプルコード（PHP）です。

CDアルバムのジャケット写真をLINEで送信すると
CDアルバムの詳細情報（タイトル/アーティスト/リリース日/レーベル）が分かります。
